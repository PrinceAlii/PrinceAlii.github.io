RateMyUni is a full-stack platform that aggregates and standardizes university subject feedback across Australia. It combines handbook scraping, verified student reviews, and search tools to help students make better unit decisions.

What I Built

  • Configurable scraping workflows for different handbook architectures (CourseLoop, search-driven, and legacy HTML).
  • A modern React frontend with fast unit lookup and filtering.
  • Verified review flows with email authentication and moderation safeguards.
  • Asynchronous ingestion pipelines using Redis job queues for bulk updates.

Stack

  • Web: React 19, Vite, Tailwind CSS, TanStack Query
  • API: Node.js, Fastify, BullMQ
  • Data: PostgreSQL, Drizzle ORM, Redis
  • Infra: Terraform, AWS (EC2, RDS, ECR), Cloudflare proxy

Outcome

  • Created a central, searchable source of course-level review data.
  • Improved data reliability through normalized scraping and moderation tooling.
  • Established a production-ready deployment pipeline with infrastructure as code.
